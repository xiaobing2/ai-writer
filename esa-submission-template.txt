【作品说明】

一、项目简介
AI 文案工作台是一个基于阿里云通义千问大模型的智能文案生成工具。通过调用阿里云 DashScope API，帮助用户快速生成各类高质量文案，包括营销文案、产品介绍、社交媒体内容、邮件模板、创意故事等。项目采用 Vue 3 + Vite 构建，部署在阿里云 ESA Pages 平台，享受全球边缘节点加速。

二、核心功能
1. 多种文案类型：支持营销文案、产品介绍、社交媒体、邮件模板、创意故事、自定义提示词等
2. AI 智能生成：基于阿里云通义千问（Qwen）大模型，生成高质量文案内容
3. 现代化 UI：简洁美观的用户界面，操作流畅，体验优秀
4. 一键复制：生成结果可快速复制到剪贴板，提高工作效率
5. 边缘加速：部署在阿里云 ESA Pages，享受全球边缘节点加速

三、技术实现
- 前端框架：Vue 3 + Vite
- AI 模型：阿里云通义千问（Qwen）系列模型
- API 接口：阿里云 DashScope OpenAI 兼容接口
- 部署平台：阿里云 ESA Pages
- 样式方案：原生 CSS + 响应式设计

四、评分维度对应说明

【创意卓越】
- 独特概念：将 AI 大模型能力与日常文案创作需求结合，解决实际工作中的痛点
- 视觉设计：采用现代化渐变色彩和卡片式布局，界面美观大方，用户体验流畅
- 交互创新：根据不同类型动态展示表单，降低使用门槛，提高生成效率

【应用价值】
- 实用性强：覆盖营销、产品、社交媒体、邮件等多种常见文案场景，即开即用
- 效率提升：快速生成高质量文案，节省创作时间，提高工作效率
- 易于扩展：模块化设计，可轻松添加新的文案类型和功能
- 部署即用：基于静态站点，部署简单，无需后端服务器

【技术探索】
- 边缘计算：充分利用 ESA Pages 的全球边缘节点，实现低延迟访问
- AI 模型集成：调用阿里云通义千问大模型，使用 OpenAI 兼容接口，技术实现规范
- 环境变量管理：通过环境变量安全管理 API Key，避免敏感信息泄露
- 响应式设计：适配桌面端和移动端，提供一致的用户体验
- 错误处理：完善的错误提示和加载状态，提升用户体验
